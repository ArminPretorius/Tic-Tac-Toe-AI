# Tic-Tac-Toe AI
## About This Project:
An AI that plays tic-tac-toe agains you using a minimax algorithm, also includes visualization of all the possible outcomes as seen by the AI.

## Languages & Tools:
- Python
- pyGame
- NetworkX
- MatPlotLib
